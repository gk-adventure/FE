# Next.js 15 + Tailwind CSS 프로젝트

이 프로젝트는 **Next.js 15 (App Router + Turbopack)** 기반이며, **Tailwind CSS**가 기본 스타일링 프레임워크로 설정되어 있습니다.

## 🔧 설치 및 실행 방법

1. **레포지토리 클론**

```bash
git clone https://github.com/gk-adventure/FE.git
cd your-repo-name
```

2. **패키지 설치**

```bash
npm install
```

3. **개발 서버 실행**

```bash
npm run dev
```

브라우저에서 http://localhost:3000 으로 접속하세요.
